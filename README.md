# APM Music Dashboard

A dark-themed music project management dashboard built with React and Vite.

## Setup

```bash
npm install
```

## Run the app

```bash
npm run dev
```

Then visit `http://localhost:5173` in your browser.

## Build for production

```bash
npm run build
```

The built files will be in the `dist` folder.

## Features

- **Header**: Logo, global search bar, notification bell (with badge), and Invite button
- **Sidebar**: Navigation icons (Home, Search, Calendar, Projects, Music, Profile)
- **Project Overview**: Image collages, description, keyword tags, metadata, "Sounds Like" feature
- **Tracks Section**: Tabs (Tracks/Albums), Reorder/Play All/Customize controls, track listing with:
  - Track number, thumbnail, title, ID
  - Versions and Stems dropdowns
  - Description and waveform visualization
  - Comment field and action icons (like, share, add, remove, Sounds like)
